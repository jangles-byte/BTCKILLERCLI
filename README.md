# BTC.KILLER CLI

```bash
git clone https://github.com/jangles-byte/BTCKILLERCLI.git && cd BTCKILLERCLI && bash setup.sh
```

After setup, type `btc-killer` from any terminal to launch.

---

**Requires:** Python 3.10+, a [Kalshi](https://kalshi.com) API key + private key, a free [Coinalyze](https://coinalyze.net) API key.
